Tags has three states (git 1.9 and later): Checked: All tags as well as branches are downloaded (--tags is passed to git), 
unchecked: No tags are downloaded (--no-tags is passed to git), and third state: use default behavior (based on remote name.tagopt setting). 
Tags has three states (prior to git 1.9): Only all tags are downloaded but no branches are downloaded (--tags is passed to git), unchecked: No tags are
 downloaded (--no-tags is passed to git), and third state: use default behavior (based on remote name.tagopt setting).
